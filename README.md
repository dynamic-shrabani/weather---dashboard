🌦️ Weather Dashboard

A modern and responsive Weather Dashboard built using HTML, CSS, and JavaScript. The application provides real-time weather information with a clean UI and interactive experience.

🚀 Live Demo

https://dynamic-shrabani.github.io/weather-dashboard/

✨ Features

- Search weather by city name
- Real-time weather data
- Temperature, humidity, and wind speed display
- Responsive design for desktop and mobile
- Modern UI with animations
- Easy-to-use interface

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Weather API
